Install the required dependencies using pip: ``` pip install Flask Flask-RESTful ``` 2. Create a `db.sqlite` database file in the root directory of the project. 3. Run the Flask application: ``` python app.py ``` 4. Visit `http://localhost:5000/` to access the frontend. 5. Visit `http://localhost:5000/api/example` to access the backend API.
